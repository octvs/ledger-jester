# ledger-jester

A stripped down fork of ledger-autosync.

## About

This small project initially started as a fork of
[ledger-autosync](github.com/egh/ledger-autosync), aiming to drop support for
`ofx` and continue with `csv` synchronization feature supported with custom
parsers. After initial personal use the codebase diverged enough to form as a
separate repository. Still a major credit goes to the original project.
